How to call this API

Headers
Authorization: Token {{ api.token }}
Content-Type: application/json
Text query — JSON body
POST {{ full_url }}
Authorization: Token {{ api.token }}
Content-Type: application/json

{"query": "{{ api.sample_query|default:'How do I treat blast in my rice crop?'|escapejs }}"}
Image-only query — multipart body
POST {{ full_url }}
Authorization: Token {{ api.token }}
Content-Type: multipart/form-data

image=<file>
Text + image query — multipart body
POST {{ full_url }}
Authorization: Token {{ api.token }}
Content-Type: multipart/form-data

image=<file>
query=Why are the leaves turning yellow on this plant?
When both are present, the image is identified by CLIP and the text question is answered using that visual context.
Audio query — multipart body (Whisper STT)
POST {{ full_url }}
Authorization: Token {{ api.token }}
Content-Type: multipart/form-data

audio=<file: .webm | .ogg | .mp3 | .m4a | .wav>
language=Hindi          # optional: overrides API default ({{ api.language }})
The audio is transcribed in the source language; the answer is returned in the same language. You can also combine audio + image in one request.
Language override: the API's default language is {{ api.language }}. Callers can override per request by sending a language field (multipart) or key (JSON). Supported: English, Hindi, Odia, Bengali, Telugu, Tamil. The answer is always returned in the request's language.

cURL — text only

curl -X POST '{{ full_url }}' \
  -H 'Authorization: Token {{ api.token }}' \
  -H 'Content-Type: application/json' \
  -d '{"query":"{{ api.sample_query|default:'rice blast treatment'|escapejs }}"}'

cURL — image only

curl -X POST '{{ full_url }}' \
  -H 'Authorization: Token {{ api.token }}' \
  -F 'image=@/path/to/photo.jpg'

cURL — text + image (combined)

curl -X POST '{{ full_url }}' \
  -H 'Authorization: Token {{ api.token }}' \
  -F 'image=@/path/to/photo.jpg' \
  -F 'query=Why are the leaves turning yellow on this plant?'

cURL — audio (with optional language override)

curl -X POST '{{ full_url }}' \
  -H 'Authorization: Token {{ api.token }}' \
  -F 'audio=@/path/to/voice.webm' \
  -F 'language=Hindi'
